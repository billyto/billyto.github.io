###Navigator search

**⌘+1** _(Command + 1)_ will land you in the project navigator then,

**⌥+⌘+J** _(Option + Command + J)_ Project Navigator Search.

###General Search

**⇧+⌘+0** _(Shift + Command + O)_ Will open a general search window.
